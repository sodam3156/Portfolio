- 웹페이지: `index.html`
- 상세 Markdown: `portfolio.md`
- 공개 연락 채널: [GitHub @sodam3156](https://github.com/sodam3156)

## 공개 원칙

- 학생번호형 이메일, 전화번호, 주소, 학적 등 개인 식별정보를 게시하지 않습니다.
- 비공개 저장소의 팀원 이름, 닉네임, 내부 예산과 운영 정보는 옮기지 않습니다.
- 실제 저장소에서 확인되는 기능·상태만 쓰고, 계획은 계획으로 표시합니다.
